# SmartDoc AI: Intelligent PDF Search Assistant 🚀

SmartDoc AI is a Retrieval-Augmented Generation (RAG) application built with Python. It allows users to upload complex PDF documents and ask questions against the content using an intelligent LLM backend.

## 🛠️ Features
* **PDF Document Ingestion:** Seamlessly parse and segment multi-page text documents.
* **Semantic Search:** Uses vector embeddings to find contextually relevant document segments.
* **Streamlit UI:** A clean, intuitive dashboard with a sidebar for secure Hugging Face API authentication.

## 💻 Tech Stack
* **Frontend:** Streamlit
* **LLM Orchestration:** LangChain
* **Model Backend:** Hugging Face Hub APIs
